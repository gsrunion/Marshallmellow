Java Binary Serialization

Problem: Java has weak built in support for exchange of structured binary data with external systems.

Binary Structured Data

typedef struct MessageHeader {
  uint8_t serviceType; // unsigned 8 bit integer
  uint8_t messageType // unsigned 8 bit integer
  uint16_t sequenceNumber;// unsigned 16 bit integer
} MessageHeader;

// in memory 0x01, 0x02, 0xFF, 0xFF
MessageHeader header = { 1, 2, 0xFFFF }
// not equivalent
public class MessageHeader {
  private byte serviceType; // signed 8 bit integer
  private byte messageType; // signed 8 bit integer
  private short sequenceNumber; // signed 16 bit integer
  ....
}

// in memory ?
MessageHeader header = new MessageHeader(1, 2, (short)0xFFFF);

Java Nuances

  • No support for unsigned types
  • Memory model is opaque to developers.
  • Primitive type sizes are well defined but not explicit in type names.
  • VM always uses same endianess. (I think).

Enter SerializableStruct …

public class MessageHeader implements SerializableStruct {
  @SerializableField byte serviceType;
  @SerializableField byte messageType;
  @SerializableField short sequenceNumber;
  @Override public int binarySize() { return 4; }
  @Override public ByteOrder defaultByteOrder() { return ByteOrder.BIG_ENDIAN; }
}

byte[] data = new byte[4]; // ...
MessageHeader header = new MessageHeader();
SerializableStructCodec.decode(header, ByteBuffer.wrap(data));

… Enter SerializableStruct

  • Handles marshalling primitive and boxed primitive types.
  • Handles marshalling of Strings
  • Handles marshalling nested SerializableStructs
  • Handles marshalling arrays of the above.
  • Gives developer control of binary memory footprint.
  • Gives developers control of endianess.

Non Structured Or Conditionally Present Data

class Struct {
    @Marshalled(codec = BooleanCodec.class) public boolean truePredicate = false;
    @Marshalled(codec = BooleanCodec.class) public boolean falsePredicate = true;
    @Marshalled(codec = ByteCodec.class, precondition = "truePredicate") public byte first;
    @Marshalled(codec = ByteCodec.class, precondition = "falsePredicate") public byte second;
    @Marshalled(codec = ByteCodec.class, precondition = "shouldInclude")  public byte third;

    public boolean shouldInclude() {
        return truePredicate;
    }
}

Variable Length Arrays And Strings

class Struct {
  @Marshalled(codec = ByteCodec.class, isArray = true) public byte[] fixedLength = new byte[3];
  @Marshalled(codec = ByteCodec.class) public byte len;
  @Marshalled(codec = ByteCodec.class, isArray = true, length = "len") public byte[] fieldPredicated = new byte[100];
  @Marshalled(codec = ByteCodec.class, isArray = true, length = "length") public byte[] methodPredicated = new byte[100];

  public int length() {
      return len;
  }
}

Nested Objects

class Inner {
  @Marshalled(codec = ByteCodec.class) public byte item;
}

class Outer {
  @Marshalled(codec = ObjectCodec.class, isArray = true)
  public Inner[] a = new Inner[] { new Inner(), new Inner() };
}
